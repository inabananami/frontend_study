js部分：
```js
	const btn = document.querySelector(".button");

	window.onscroll = function() {
		//获取页面滚动位置
		if(document.documentElement.scrollTop > 100) {
			btn.style.display = "block";
		}
		else {
			btn.style.display = "none";
		}

		//点击按钮时滚动回页面顶部
		btn.addEventListener("click", function() {
			window.scrollTo({
				top: 0,
				behavior: "smooth"
			});
		});
	}
```
### 2. 原理说明
- 使用`document.documentElement.scrollTop`来检测页面有没有返回顶部，其数值是像素值。当上述值大于100（鼠标滑动一次），就显示返回页面的按钮。否则不显示。
- 再给这个返回按钮加一个click事件监听，执行scrllTo函数（使页面离最顶端0像素，变化为平滑）。就可以实现由快到慢的返回页面顶部了。

```js
window.addEventListener("scroll", function() {
	let scrollTop = document.documentElement.scrollTop;
	let scrollHeight = document.documentElement.scrollHeight;
	let clientHeight = document.documentElement.clientHeight;

	if(scrollTop + clientHeight >= scrollHeight - 50) {
		console.log("检测到用户下拉，加载更多内容...");
		const img_box = document.createElement("div");
		const root_box = document.querySelector(".root_box");
		img_box.classList.add("img_box");
		img_box.innerHTML += `            
		<img src="../img/ayumu3.1.jpg">
		<img src="../img/BanGDream.png">
		<img src="../img/nijigasaki.png" alt="">
		<img src="../img/osu.jpeg" alt="">
		<img src="../img/ayumu3.1.jpg" alt="">
		<img src="../img/BanGDream.png" alt="">
		`;
		root_box.append(img_box);
	}
});
```
### 2. 原理说明
- 检测页面是否到达顶部这个代码同页面返回顶部的原理，一样使用了`document.documentElement.scrollTop`来检测页面有没有返回顶部，其数值是像素值。`document.documentElement.scrollHeight`是目前划过页面的高度；`let clientHeight = document.documentElement.clientHeight` 是目前静态页面的高度。
- 检测到用户滑动到了最底部，创建一个新div，然后给他加root_box类，最后把相应的img加到div里面。（在正经的项目中，应该是调取数组中的图片链接，然后打印到页面中）。

### 2. 原理说明
- 主要功能：可以自动设置图片偏移量，自动算移动量，程序复用性高。
- 原理：
 1. 长盒子偏移量公式 = `(spot_array.length - 1)\*415px`，因为我们要让第一张图片居中，就得让图片向右移动一半的位置。
 2. 封装小红点动作至`dot_pos(j)`,意思为：清除其他dot样式，然后将对应的小红点加上active类名。
 3. **左箭头原理**：当j（图片下标）不为0时，则j--，translatex中的像素为`(-830 \* j)px`；当j为0时，则图片直接移动`(-830 \* (j + spot_array.length - 1))px`，即图片末尾。动画为`translate 0.5s ease-in-out`。
 4. **右箭头的原理**比较简单：当j为`array_spot.length - 1`时，直接让长盒子回到原点（0px）；其他情况则移动`(-830 \* j)px`。
 5. 小圆点 **(dot(i)函数)** 则使用了类似于右滑的原理：先加上dot_pos(j)函数。当点击到第(i+1)个小圆点时，先将这个小圆点的信息更新给j，然后将长盒子的位置移动到j对应的图片位置，即还是`(-830 \* j)px`。
 6. 左右箭头默认为消失状态`opacity: 0`,`array_appear()`状态即修改两个箭头到`opacity: 1`,动画为`opacity 0.3s ease-in-out`。
 7. 依次给左右array加上`click`事件，套上`left_slide`和`right_slide`函数。使用for循环遍历小圆点数组，执行dot(i)函数。
 8. 默认设定定时器，套上`right_slide`函数
 9. 给`.carsel_inner_box`加上`mouseover`与`mouseleave`事件，前者清除计时器，再调用`array_appear()`,后者先清除计时器再定义与之前相同的计时器，然后再让箭头消失。
 10. 小圆点原理同上，只不过不需要控制箭头的出现与消失。
